Accounting Topic Evolvement - A Glimpse at FAF Twitter Account¶

The FAF (Financial Accounting Foundation) consists of 2 departments, one is FASB (Financial Accounting Standard Board) which is responsible for establishing financial accounting and reporting standards for public and private companies, the other is GASB(Goverment Accounting Standard Board) which establishes standards for U.S. state and local governments. In this blog, we take a brief look at the FAF twitter account and see accouting topics developed in both FASB and GASB over the past few years.

Collecting Data From Twitter¶

The data is retrieved as of June 1, 2018. To retrieve data relevant to the FAF account @FAFNorwalk from twitter, I first registered my application on http://apps.twitter.com to obtain a consumer key, consumer secret, access token and an access token secret so that my application can get an access to Twitter. I then used Python libary Tweepy to get the data of the followers(people who are following the FAF), friends of the FAF(people whom the FAF are following), and all tweets posted by the FAF account. The acquired data is stored in the respective csv files.

Tweets Hashtag Analysis¶

The tweets dataframe has a column "hashtags" that stores a list of hashtags for each tweet. I want to do more granular analysis on hastags, so I split hashtags for each tweet and have one hastag per row.

In [27]:
# create a series with 2-level index, and each item for a single hashtag
hashtag_series = tweets_df['hashtags'].str.split(' ').apply(pd.Series, 1).stack()
# drop the inner level of index to align the original index in the tweets dataframe
hashtag_series.index = hashtag_series.index.droplevel(-1)
# assign a name to the series, this is required by the join operation
hashtag_series.name = 'hashtag'
# join the tweets dataframe with the column "created_at" to the hashtag series by index
hashtag_df = tweets_df.loc[:, ['created_at']].copy().join(hashtag_series)
hashtag_df.dropna(axis=0, how='any', inplace=True)
